# portipy
A simple port scanning script.

## Requirements:
* python-nmap: install via ```pip install python-nmap```

## Usage:
1. ```$ python porti.py```
2. Enter hostname
3. Enter ports to scan, can be separated by ',' or '-' to scan a range of ports

## Nice features:
* Colored output!
